# Agent feedback

Ce dossier sert à déposer les comptes-rendus produits par des agents (humains ou IA) qui travaillent en parallèle sur les tâches du dossier [Plan](../Plan).

## Règles
- 1 rapport = 1 fichier Markdown.
- Commencer depuis le template: [REPORT-TEMPLATE.md](./REPORT-TEMPLATE.md)
- Nommer les fichiers de façon triable: `YYYY-MM-DD__agent__sujet.md`
  - Exemple: `2025-12-18__payments-agent__dexchange-webhooks.md`

## Attendu
- Un résumé exécutable (décisions proposées).
- Les hypothèses explicitement listées.
- Les liens sources et/ou fichiers du repo concernés.
